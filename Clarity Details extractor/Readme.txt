1. Download Billing Report generator.xlsm and Timesheet Extract .xlsx to your local copy
2. Open Billing Report generator.xlsm
3. Click on the button with Maintainence Icon
4. Click Browse and locate the Timesheet Extract .xlsx file
5. Click Add Salary ID, you can click this as many times you want
6. Select from the salary ID
7. Select the projects code for the salary ID for which you want the billing to be calculated
8. Repeat for as many salary ID's you want this for.
9. Click Save icon and close it
10. Click on Fetch button

Voila!! You should get the billing details of your resources
